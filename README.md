

#googleByShell

A simple tool to search in google/baidu/wikipedia/etc by shell

##Version 1.0.0

	Search in google 
	Search in baidu
	Search in wikipedia

##Install 

	cd into the folder
	chmod +x ./setup.sh
	./setup.sh

##Usage
	
	$ google hello
	$ baidu hi
	$ wiki hi
	$ google --version
	$ google --help


##Have fun!
